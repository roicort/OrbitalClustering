# Órbitas

Sarajilic et al. propusieron extender la idea una firma orbital a grafos dirigidos [Sar+16]. Dada la cantidad de posibles configuraciones para las órbitas en un graphlet dirigido, los autores limitan el conteo a graphlets de hasta 4 nodos. Con estas consideraciones, la firma orbital resultante para cada nodo es un vector en R129 donde el componente i representa el conteo de la órbita i, de acuerdo a la descripción presentada en la Fig. 4.1.
